# create-your-own-php-framework
构建自己的PHP框架

这个栏目暂时告一段落，前一部分比较像Yii，后一部分比较像Laravel，因为当时正在看相应框架的源码，所以会有不少借鉴参考。捂脸～

这个框架千万不要直接应用于生产环境，只是用来帮助大家理解PHP框架的实现机制。
